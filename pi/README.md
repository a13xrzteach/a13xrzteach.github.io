# Server
## Dependencies
- Standard Linux installation
- ``git``
- ``jq``
- ``darkhttpd``

## Startup
Disable power saving and automatic screen locks in KDE settings

```
cd a13xrzteach.github.io/pi
./server
```

Update the upstream ``./endpoint`` to match the location of the server
